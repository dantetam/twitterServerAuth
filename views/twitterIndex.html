Hello, this is a full-stack mongo.db, express, node.js server for experimentation with async node.js, RESTful APIs, and MongoDB storage and querying.


Start up the Twitter server. /twitter   With topic parameter. /twitter/:topic
Search a topic and dynamically deliver a word map visualization. /twitter/wordmap/:topic
Search live tweets and group by topic associations. /twitter/topicgroups



Search the tweet data. /twitterData   With query parameter. /twitterData/:query
Search most recent tweets. /twitterData/recent
Search most recent tweets and use cluster approximation algorithm. /twitterData/recentcluster
Search most recent tweets and use MST algorithm. /twitterData/recentmst
Better topic grouping algorithm (overlap). /twitterData/topicgroups
Query a large amount of tweets and visualize their words. /twitterData/wordmap
Analyze tweets for words after a chosen topic. /wordlookup/:searchTweetWord/:inspectWord

Get current database status. /count  
Test the login screen and database. /login
